# LLM Gateway Admin Web

This frontend is built with React, TypeScript, Vite, TanStack Query, Zustand, Tailwind CSS, and shadcn/ui-style local components.

## Local Development

```bash
npm install
npm run dev
```

The Vite dev server runs on port `5173` and proxies `/admin-api` to `http://localhost:9000`.

## Build

```bash
npm run build
```
